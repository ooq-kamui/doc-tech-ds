 
# [ ] からはじまる コマンド


```
[ctr-d  カーソル位置の単語と一致する #define の文を
        カレントファイルとインクルードファイルの中から探し
        最初に見つけた場所へジャンプする
        カレントファイルの先頭から検索する
[ctr-o  カーソル位置の単語を含む行をカレントファイルの先頭から探し
        最初に見つかった行へジャンプする
[#      後方へ対応のとれていない #if, #else, #ifdef を探し
        N 個目に見つかった場所へジャンプ
['      後方の小文字マークがある行の最初の非空白文字へ移動する
[(      後方へ対応のとれていない '(' を探し
        N 個目に見つかった場所へジャンプする
[*      [/ と同じ
[`      後方の小文字マークがある行へ移動する
[/      後方へ C コメントの始まりを探し
        N 個目に見つかった場所へジャンプする
[D      カーソル位置の単語にマッチする単語の定義を
        カレントファイルとインクルードファイルの中から検索し一覧表示する
        カレントファイルの先頭から検索を開始する
[I      カーソル位置の単語にマッチする単語を含む行を
        カレントファイルとインクルードファイルの中から検索し一覧表示する
        カレントファイルの先頭から検索を開始する
[P      [p と同じ
[[      カーソルを N 段落後方へ移動
[]      カーソルを N SECTION 後方へ移動
[c      カーソル後方へ N 個目の変更箇所へ移動
[d      カーソル位置の単語にマッチする単語の最初の定義(#define)を
        カレントファイルとインクルードファイルの中から検索し表示する
        カレントファイルの先頭から検索を開始する
[f      gf と同じ
[i      カーソル位置の単語にマッチする単語を含む最初に見つかった行を
        カレントファイルとインクルードファイルの中から検索し一覧表示する
        カレントファイルの先頭から検索を開始する
[m      後方へ N 番目に現れた関数の先頭へカーソルを移動
[p      P と同様、さらに現在の行と同じインデントにする
[s      前のスペルミスのある単語へ移動
[z      開いている折り畳みの最初の行へ移動
[{      後方へ N 番目に現れた対応する括弧のない '{' へカーソルを移動


]ctr-d  カーソル位置の単語にマッチする単語の最初の定義(#define)を
        カレントファイルとインクルードファイルの中から検索しジャンプする
        カーソル位置から検索を開始する
]ctr-o  カーソル位置の単語にマッチする単語を含む最初に見つかった行を
        カレントファイルとインクルードファイルの中から検索し
        ジャンプする
        カーソル位置から検索を開始する
]#      前方の対応の取れていないN個目の #endifか #else へ移動
]'      前方の小文字マークがある行の最初の非空白文字へ移動する
])      前方の対応の取れていないN個目の ')' へ移動
]*      ]/ と同じ
]`      前方の小文字マークがある行へ移動する
]/      カーソルを N 先の C のコメントの最後へ移動
]D      カーソル位置の単語にマッチする単語の定義 (#define) を
        カレントファイルとインクルードファ イルの中から探して表示する
        カーソル位置から検索を開始する
]I      カーソル位置の単語にマッチする単語を含む行を
        カレントファイルとインクルードファイルの中から探して表示
        カーソル位置から検索を開始する
]P      [p と同じ
][      カーソルを N SECTION 前方へ移動
]]      カーソルを N 段落前方へ移動
]c      カーソル前方へ N 個目の変更箇所へ移動
]d      カーソル位置の単語にマッチする単語の最初の定義(#define)を
        カレントファイルとインクルードファイルの中から検索し表示する
        カーソル位置から検索を開始する
]f      gf と同じ
]i      カーソル位置の単語にマッチする単語を含む最初に見つかった行を
        カレントファイルとインクルードファイルの中から検索し一覧表示する
        カーソル位置から検索を開始する
]m      前方へ N 番目に現れた関数の末尾へカーソルを移動
]p      p と同様、さらにカレントラインにインデントをあわせる
]s      次のスペルミスのある単語へ移動
]z      開いている折り畳みの最後の行へ移動
]}      カーソルを N 先のマッチしていない '}' へ移動
```

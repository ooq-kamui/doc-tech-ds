
# keybind `<c-w>`

## keybind `<c-w>`

```
<C-w> c : カレントウィンドウを閉じる
<C-w> q : カレントウィンドウを閉じる（内容が保存されていない場合は警告する）
<C-w> = : 複数のウィンドウのサイズを等しくする
<C-w> w : 次のウィンドウに移動する
<C-w> h : 左隣のウィンドウに移動する
<C-w> j : 下のウィンドウに移動する
<C-w> k : 上のウィンドウに移動する
<C-w> l : 右隣のウィンドウに移動する
<C-w> v : ウィンドウを垂直に分割する
<C-w> s : ウィンドウを水平に分割する
<C-w> _ : 現在のウィンドウを垂直方向に最大化する
<C-w> | : 現在のウィンドウを水平方向に最大化する 

<C-w> z : ウィンドウを解放し、そのウィンドウのタブだけを表示する
<C-w> t : 現在のウィンドウを一番上のタブの先頭に移動する 
```

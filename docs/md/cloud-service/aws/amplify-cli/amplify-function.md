
# amplify function


## function 追加

```
amplify function add
```

or

```
amplify add function
```


## function 個別 削除 ( remove )

```
amplify function remove function01
```

で function 個別削除 できる


function name 引数指定なし

```
amplify function remove
```

だと, リストから選択 となる

amplify push して反映

```
amplify push function function01
```


console で確認

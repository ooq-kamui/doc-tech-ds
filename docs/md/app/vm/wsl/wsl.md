
# wsl


## wsl 自体を install

microsoft store を使用しない

```
wsl --install --web-download
```


## wsl 自体を update

```
wsl --update --web-download
```


## install できる distro を表示

```
wsl --list --online
```


## distro を install

```
wsl --install -d <distro>
```


## install されている distro を表示

```
wsl --list --verbose
```

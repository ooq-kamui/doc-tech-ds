
# brew ( homebrew )


## brew 自体の install

hp のトップページに書いてある install cmd を実行する

https://brew.sh/ja/


上記の cmd 実行後, メッセージの最後に表示される cmd 実行する


## brew 自体の update

```
brew update
```


## brew 自体の uninstall

```
wip:
```


## cask とは

gui の app を install など するもの


## package ( app ) list

```
brew list
```


## package ( app ) の update

```
brew upgrade xxx
```


## package ( app ) の uninstall

```
brew uninstall xxx
```


# tar


## 解凍

### auto

```
tar -xvf xxx.tar.yy
```

### tar

```
tar -xvf xxx.tar
```

### tar.gz

```
tar -zxvf xxx.tar.gz
```


## 圧縮

### tar

```
tar -cvf xxxx.tar directory
```

### tar.gz

```
tar -zcvf xxxx.tar.gz directory
```


## option

```
-v  実行状況 を表示 view
-f  圧縮ファイル名  file

-x  解凍            ??

-c  圧縮            cre
```

### 圧縮形式

```
-z  gz
-j  bz2
-J  xz
```

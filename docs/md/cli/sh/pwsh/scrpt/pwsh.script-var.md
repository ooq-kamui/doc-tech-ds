
# var


## bool

bool は自動変数で定義されている

```
$true   : true
$false  : false
```

ref
https://learn.microsoft.com/ja-jp/powershell/module/microsoft.powershell.core/about/about_automatic_variables?view=powershell-7.4


## scope

### local var

```
$local:val = 1
```

### global var

```
$global:val = 1
```

### env var

```
$env:val = 1
```

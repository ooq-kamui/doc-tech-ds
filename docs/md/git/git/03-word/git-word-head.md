
# HEAD


## HEAD

- `HEAD` はその branch の最新の ( 先端の ) commit のこと


## `HEAD~`

- `HEAD` の 1つ前の commit のうち, 1つ目の 親 commit


## `HEAD~~`

- `HEAD` の 2つ前の commit のうち, 1つ目の 親 commit


## `HEAD^`

- `HEAD` の 1つ前の commit のうち, 1つ目の 親 commit
- `HEAD~` と同じ
- `HEAD^1` の省略形 ( と考えられる )


## `HEAD^2`

- `HEAD` の 1つ前の commit のうち, 2つ目の 親 commit


## `HEAD^^`

- `HEAD` の 1つ前の commit のうち, 1つ目の 親 commit  
    の 1つ前の commit のうち, 1つ目の 親 commit
